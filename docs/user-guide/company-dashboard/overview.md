# Company Dashboard Overview

**URL:** `/dashboard`

The dashboard home gives a snapshot of your business activity on the platform.

## What you see

| Widget / section | Description |
|------------------|-------------|
| **Summary metrics** | Total chats, orders, revenue, messages (period-based) |
| **Recent chats** | Latest customer conversations with unread indicators |
| **Recent orders** | Newest orders with status badges |
| **Notifications** | In-app alerts for new messages, orders, subscription reminders |
| **Quick links** | Shortcuts to Chats, Orders, Products, Settings |

## Navigation sidebar

| Menu item | Purpose |
|-----------|---------|
| Dashboard | Home overview |
| Chats | WhatsApp inbox |
| Orders | Order management |
| Products | Product catalog |
| Customers | Customer directory |
| FAQs | Knowledge base |
| Analytics | Reports and charts |
| Growth | Growth Engine |
| Subscription | Plan, usage, billing |
| Settings | Business, AI, WhatsApp, payments |

## Notifications

- Bell icon shows unread count
- Click to view notification list
- Mark individual or all as read
- Notifications trigger for: new customer messages, new orders, subscription events

## Mobile experience

The dashboard is responsive. On mobile, use the hamburger menu to access sidebar navigation.

## Prerequisites for full functionality

| Feature | Requirement |
|---------|-------------|
| Bot auto-replies | Active subscription + WhatsApp connected + auto-reply enabled |
| Order notifications | WhatsApp connected |
| Growth Engine | Active subscription + plan includes Growth limits |
| Payment collection | Order payment settings configured |
